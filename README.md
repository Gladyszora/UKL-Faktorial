# Menghitung Faktorial
​Fungsi Program: Untuk menghitung nilai faktorial (n!) dari suatu bilangan bulat positif yang dimasukkan oleh pengguna.
​Cara Kerja: Program mengambil bilangan bulat positif n, lalu menggunakan perulangan (loop) untuk mengalikan semua bilangan bulat dari n hingga 1. Hasil perkalian tersebut disimpan dalam variabel bertipe long untuk menampung nilai yang besar, dan kemudian dicetak sebagai output.
